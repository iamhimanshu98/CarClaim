Overview
Welcome to CarClaim: Detecting Insurance Fraud. An intelligent system designed to detect potential car insurance fraud. Whether you’re an insurance professional, a policyholder, or an investigator, our platform provides valuable insights into the legitimacy of car claims.

Technology Stack
Frontend : An intuitive user interface using HTML and CSS, ensuring an appealing design that’s easy to navigate.
Backend : Runs on Python and Flask, allowing seamless communication between the frontend and the machine learning model.
Machine Learning : Detection model leverages a Decision Tree algorithm, analyzing claim data to identify suspicious patterns.

How it works
Input Data : Users submit essential information related to car claims, accident details, policy history, and claimant profiles.
Fraud Detection : ML model processes this data, flagging potentially fraudulent claims based on historical patterns and records.
Result : Instantly tells a claim is fraud or not, empowering insurance professionals to make informed decisions.
